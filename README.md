# Flask Web Calculator

A simple and clean web-based calculator built using **Python**, **Flask**, and **HTML**.  
This project started as a basic Python terminal calculator and was upgraded into a fully functional web app to showcase backend development skills and web integration.

---

## 🚀 Features
- Perform basic arithmetic operations: addition, subtraction, multiplication, division  
- Clean and minimal web interface  
- Error handling (e.g., division by zero)  
- Built with Flask using a simple and readable structure  
- Great beginner-friendly example of turning a Python script into a web application

## 🧠 How It Works
- The user enters two numbers and selects an operator  
- The form sends the data to the Flask backend  
- Flask processes the math logic  
- The result is rendered back onto the webpage
